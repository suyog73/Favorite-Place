# favorite_places

An android app where you can pick image and pick location and store in local database.

## Packages

- flutter_riverpod => For state management
- image_picker => To capture image from native device 
- location => To get latitude and longitude for selected location
- geocoding => To convert lat and lng into readable address
- flutter_map => To show map preview and select location from map
- path_provider & path => To manage local directory paths
- sqflite => To create SQL database on local device and store data locally
